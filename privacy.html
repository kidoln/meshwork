MeshWork 隐私政策

最后更新：2026年5月

概述

MeshWork 是一款本地优先的创作工作台应用。我们深知隐私的重要性，并致力于保护您的数据安全。本隐私政策说明了我们如何处理您的数据。

核心原则：您的数据存储在您的设备上，您完全掌控自己的信息。

我们收集哪些信息

1. 本地数据

MeshWork 在您的本地设备上存储以下数据：

  • 您创建的文档和笔记内容
  • 讨论区的消息和话题
  • 应用设置和偏好配置
  • AI 提供商配置；API 密钥存储在 macOS 钥匙串中
  • 网络搜索提供商配置；搜索服务 API 密钥存储在 macOS 钥匙串中
  • Git 仓库配置；令牌、口令和粘贴的 SSH 私钥内容存储在 macOS 钥匙串中
  • 网络代理配置；代理密码存储在 macOS 钥匙串中

所有这些数据都存储在您的本地设备上，MeshWork 不会将这些数据上传到我们的服务器。

2. 网络活动

当您使用 AI 功能或网络搜索功能时，应用会向第三方服务发送请求。这些请求直接发送到您配置的服务提供商：

  • AI 服务提供商（如 OpenAI、Claude 等）
  • 网络搜索服务提供商（如 Brave Search、Serper、Tavily 等）
  • Git 仓库托管平台（如 GitHub）

这些网络请求直接在您的设备和第三方服务之间进行，MeshWork 不充当中介或代理，我们不会记录或访问这些通信的内容。

我们如何使用您的信息

MeshWork 不会收集、存储或使用您的数据用于任何商业目的。您的数据：

  • 仅存储在您的本地设备上
  • 仅用于您在应用中的直接操作
  • 仅在您主动启用时才会通过 Git 同步到您指定的远程仓库
  • 仅在您主动调用时才会发送到您配置的 AI 或搜索服务

数据共享

MeshWork 不会与第三方共享您的数据，除非：

  • 您主动选择将数据通过 Git 推送到远程仓库
  • 您主动使用 AI 功能，将内容发送到您配置的 AI 服务提供商
  • 您主动使用网络搜索功能，向搜索服务提供商发送查询请求

在这些情况下，数据传输直接发生在您的设备与您选择的服务之间，MeshWork 不会访问或存储传输的内容。

数据安全

我们采取以下措施保护您的数据安全：

  • 应用保存的 API 密钥、Git 令牌/口令/私钥内容和代理密码存储在 macOS 钥匙串中
  • 服务名称、模型、URL、端口、用户名和界面偏好等非敏感配置仅存储在本地
  • 应用运行在 macOS 应用沙盒中，限制了文件系统访问
  • 网络通信仅用于您明确配置的服务

儿童隐私

MeshWork 不面向 13 岁以下儿童。如果我们发现无意中收集了儿童的个人信息，将采取措施删除该信息。

隐私政策的变更

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，并在页顶标注更新日期。

联系我们

如果您对本隐私政策有任何疑问或建议，欢迎通过以下方式联系：

  • 在 GitHub 上提出 Issue 或 Pull Request
  • 通过应用内的反馈功能联系我们

MeshWork Privacy Policy

Last Updated: May 2026

Overview

MeshWork is a local-first creative workspace application. We understand the importance of privacy and are committed to protecting your data. This privacy policy explains how we handle your data.

Core Principle: Your data stays on your device. You are in complete control of your information.

Information We Collect

1. Local Data

MeshWork stores the following data locally on your device:

  • Documents and notes you create
  • Discussion messages and topics
  • App settings and preferences
  • AI provider configurations; API keys are stored in the macOS keychain
  • Web search provider configurations; search API keys are stored in the macOS keychain
  • Git repository configurations; tokens, passphrases, and pasted SSH private key content are stored in the macOS keychain
  • Network proxy configurations; proxy passwords are stored in the macOS keychain

All this data is stored on your local device. MeshWork does not upload this data to our servers.

2. Network Activity

When you use AI features or web search features, the app sends requests to third-party services. These requests are sent directly to the service providers you configure:

  • AI service providers (such as OpenAI, Claude, etc.)
  • Web search providers (such as Brave Search, Serper, Tavily, etc.)
  • Git hosting platforms (such as GitHub)

These network requests occur directly between your device and the third-party services. MeshWork does not act as an intermediary or proxy, and we do not log or access the content of these communications.

How We Use Your Information

MeshWork does not collect, store, or use your data for any commercial purposes. Your data:

  • Is stored only on your local device
  • Is used only for your direct actions within the app
  • Is synced to remote repositories via Git only when you explicitly enable it
  • Is sent to AI or search services only when you actively invoke them

Data Sharing

MeshWork does not share your data with third parties, except when:

  • You choose to push data to a remote repository via Git
  • You actively use AI features, sending content to your configured AI service provider
  • You actively use web search features, sending queries to your search provider

In these cases, data transfer occurs directly between your device and the services you choose. MeshWork does not access or store the transmitted content.

Data Security

We take the following measures to protect your data:

  • API keys, Git tokens/passphrases/private key content, and proxy passwords saved by the app are stored in the macOS keychain
  • Non-sensitive settings such as service names, models, URLs, ports, usernames, and interface preferences are stored locally
  • The app runs within the macOS application sandbox, limiting file system access
  • Network communication is only used for services you explicitly configure

Children's Privacy

MeshWork is not directed at children under 13 years of age. If we become aware that we have inadvertently collected personal information from a child, we will take steps to delete that information.

Changes to This Privacy Policy

We may update this privacy policy from time to time. Updated policies will be posted on this page with the revision date noted at the top.

Contact Us

If you have any questions or suggestions about this privacy policy, please contact us:

  • Submit an Issue or Pull Request on GitHub
  • Use the in-app feedback feature to contact us
